# BiblioIA
BiblioIA: Sistema de Gestión de Biblioteca con Agente IA. Trabajo Práctico Integrador de Base de Datos - ISI (UTN FRCU).
Sistema de Gestión de Biblioteca que integra una base de datos relacional robusta con un Agente de Inteligencia Artificial. 
Este proyecto permite a los usuarios interactuar con el catálogo de la biblioteca formulando preguntas en lenguaje natural (español), las cuales son traducidas automáticamente a consultas SQL ejecutables (Text-to-SQL). Además, incluye un módulo de recomendaciones personalizadas basadas en el historial de préstamos del usuario.
Desarrollado como Trabajo Práctico Integrador para la materia Base de Datos de 3º año de la carrera Ingeniería en Sistemas de Información, UTN FRCU.

Tecnologías utilizadas:
* MySQL 8+
* Python 3.10+ (Jupyter Notebook)
* Gemini 2.5 Flash
